Tutorial : Post Effect Toggle

Objects: VPostEffectToggleTrack, VPostEffectToggleEvent

A Post Effect Toggle track must be added to a Camera Group and will allow you to enable or disable a Post Effect.


How to Use:

To toggle a Post Effect, you must first create a valid Camera Group.If you are unsure of how to create or validate a Group, click here.

Right-Click on the Group, go to "Add Track" and select "Add Post Effect Track".

Right-Click on the Track and select "Add Event".

Select the newly created Event. The Property List window will refresh and inspect the Event. Update the "Action" property to determine whether to turn the referenced object ON or OFF once the Event is triggered.
